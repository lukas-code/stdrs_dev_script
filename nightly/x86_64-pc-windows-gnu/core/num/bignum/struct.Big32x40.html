Stack-allocated arbitrary-precision (up to certain limit) integer.

This is backed by a fixed-size array of given type (â€œdigitâ€). While the array is not very large (normally some hundred bytes), copying it recklessly may result in the performance hit. Thus this is intentionally not Copy.

All operations available to bignums panic in the case of overflows. The caller is responsible to use large enough bignum types.
