Slice selection

This module contains the implementation for slice::select_nth_unstable. It uses an introselect algorithm based on Orson Petersâ€™ pattern-defeating quicksort, published at: https://github.com/orlp/pdqsort

The fallback algorithm used for introselect is Median of Medians using Tukeyâ€™s Ninther for pivot selection. Using this as a fallback ensures O(n) worst case running time with better performance than one would get using heapsort as fallback.
